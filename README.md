# 一个自己用的闹钟app
功能描述
1. 闹钟功能
   1. 必须实现和闹钟相同功能的闹钟功能，在特定的时间以特定的铃声响起，并且可以关闭或者延后五分钟
   2. 需要有无声闹钟，就是和闹钟完全相同的功能，唯一的变化就是没有声音
   3. 支持备注
2. 实现当闹钟被关掉之后可以跳出一条便利贴，便利贴是随机推送一条你自己提前写好的备忘/目标，然后如果这个目标被设置过时间，那么同时展示还有多少天到期；便利贴也可以是随机的一句治愈的话或者美好的话语；也可以是今日天气
   1. 便利贴弹出用卡片动画
3. 心情打卡：关掉闹钟后弹出"今天几分"，一键记录，月底看趋势
4. 记录每天早起的时间（可以每天早上六点钟开始，从六点钟开始记录第一次解锁锁屏的时间，记录为当天起床时间），月底可以总结出一张图，具体如何呈现暂时不管
5. UI做好看一点，但是不要太多emoji
6. 可以做成卡片流
    关闭闹钟
    → 生日提醒（若有）
    → 便利贴卡片（看一眼，可以标记目标完成）
    → 今日小任务（一键完成 / 换一条 / 自定义任务池）
    → 上下滑动切换
    → 心情打卡（点一下分数）
    → 进入主界面
7. 朋友生日提醒（单独有一个管理页面）
   1. 可以添加朋友的生日（姓名 + 日期）
   2. 每个生日可以添加多条提醒，每条提醒包含：
      - 提前几天触发（比如提前7天、提前1天）
      - 这条提醒要做什么（自由填写，比如"订蛋糕"、"准备礼物"、"发消息"）
   3. 提醒触发方式：当天早上闹钟关掉后，在卡片流中插入一张生日提醒卡片
   4. 生日当天卡片样式和普通提醒卡片有所区分

## Android 工程（Kotlin + Jetpack Compose）

本目录已包含可直接用 **Android Studio** 打开构建的 Gradle 工程，应用 ID：`com.nierduolong.morningbell`，产品名：**你尔多龙吗**。

### 如何构建 / 安装到小米手机

1. 安装 [Android Studio](https://developer.android.com/studio)，首次打开会自动拉取 Android SDK。
2. 用 Android Studio **Open** 本文件夹根目录（含 `settings.gradle.kts`）。
3. **一键打包发 APK**：在项目根目录执行 `./package_apk.sh`，会在 `dist/你尔多龙吗.apk` 生成安装包（默认先 `clean` 再构建，避免 KSP 偶发失败；要快可加 `./package_apk.sh --fast`）。
4. 若手动命令行构建：执行 `./gradlew :app:assembleDebug`，APK 位于  
   `app/build/outputs/apk/debug/app-debug.apk`，拷到手机安装即可（侧载）。  
   Room 已通过 KSP **生成 Kotlin**（`room.generateKotlin`），避免旧版 Java 生成物在增量构建里叠加导致 **`AppDatabase_Impl` 类重复**；若仍异常请先 `./gradlew clean`。
5. **首次在本机命令行构建**需要 `local.properties` 中的 `sdk.dir` 指向你的 SDK（仓库里有一份示例路径；若与你的电脑不一致请自行修改）。
6. **小米 HyperOS**：请在系统设置里为该应用开启 **自启动、后台无限制、精确闹钟** 等，否则可能被推迟或杀后台。
7. **早起统计**：应用会以 **低打扰前台服务**（通知「早起统计运行中」）保持进程，用于注册 `USER_PRESENT` 并在「统计起始时刻」之后写入当日首次解锁时间；开机、进入首页或闹钟触发时会自动拉起该服务。首页若提示 **忽略电池优化**，建议按引导操作以提高留存（无法绕过用户「强制停止」或厂商彻底禁用自启动）。
